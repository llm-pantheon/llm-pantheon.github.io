    # gpt-oss-120b

    
OpenAI · open weights released 5 Aug 2025 · Apache 2.0
    
The larger of the two open-weight models OpenAI released on 5 August 2025 — its first open weights since [GPT-2](../gpt-2/) in 2019. A ~117B-parameter mixture-of-experts reasoner (~5.1B active) sized to run on a single 80GB GPU, released alongside the smaller [gpt-oss-20b](../gpt-oss-20b/).

## Official record

    

      
- Released 5 August 2025, open weights, Apache 2.0 — OpenAI’s first open-weights release since GPT-2 (Nov 2019).
      
- As published: ~117B-parameter MoE, ~5.1B active per token; positioned as o4-mini-class on reasoning benchmarks; fits a single 80GB GPU; adjustable reasoning effort; the “harmony” response format.
      
- tk — benchmark table as published, exact context window, safety-evaluation framing (the announcement discussed worst-case-fine-tuning testing).
